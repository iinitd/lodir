# LODIR

mkdir and localize it in macOS.

# INSTALL

```sh
sudo sh -c "curl https://raw.githubusercontent.com/cogons/lodir/master/lodir -o /usr/local/bin/lodir && chmod +x /usr/local/bin/lodir"
```
